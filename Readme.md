### On this project I used and learned:
* **JavaScript** (ES 2015+)
* ECMAScript modules
* DOM
* HTML5 Validation API
* Leaflet
* Fetch API, Promise, JSON

### To run this project locally:

#### 1. Clone branch `master` with the `git@github.com:NaviSparrow/1778515-keksobooking-23.git`

#### 2. `npm install`

#### 3. `npm run start`
